# Demo Car Racing Game

A simple **car racing game** built with **Python and Pygame**, fully self-contained with **demo cars drawn using shapes**. No external images or sound files are required—just run the Python file and play!

---

## Features

- **Player Car**: Demo car with colored roof and wheels.  
- **Enemy Cars**: Multiple demo cars moving down as obstacles.  
- **Road**: Includes left/right boundaries and center dashed line.  
- **Scoring System**: In-memory score and high score tracking.  
- **Collision Detection**: Game over when player collides with an enemy.  
- **Restart Option**: Press `R` to restart the game after a crash.  
- **Fully Self-Contained**: No folders, images, or sound files required.  

---

## Controls

- **Left Arrow** → Move car left  
- **Right Arrow** → Move car right  
- **R** → Restart after game over  
- **Close Window** → Quit game
